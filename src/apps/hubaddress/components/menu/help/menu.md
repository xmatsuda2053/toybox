# メニュー (MENU) の使い方

メニューは、画面の左端に配置されているアイコン群です。  
検索状態の初期化や、登録されているデータの管理を行うための機能を提供します。

## 1. メニュー項目の説明

上から順に以下のボタンが配置されています。

### 1.1. Clear（クリア）

一番上の消しゴムアイコンのボタンです。  
クリックすると、職員情報および組織情報に入力されている検索キーワードをすべてクリア（初期化）します。  
素早く全体表示に戻したい場合に便利です。

### 1.2. Delete All（データ削除）

中央の四角形に斜線が入ったアイコンのボタンです。  
アプリに登録されているすべての職員情報と組織情報を一括で削除します。  
データの更新が必要な場合や、不要になったデータを整理する場合に使用します。

クリックすると確認のダイアログ（登録データ削除）が表示されます。  
「削除」ボタンをクリックすると、データは完全に消去され、元に戻すことはできません。

### 1.3. Help（ヘルプ）

一番下の「？」アイコンのボタンです。  
クリックすると、このヘルプ画面（マニュアル）が表示されます。  
各機能の使い方を確認できます。
