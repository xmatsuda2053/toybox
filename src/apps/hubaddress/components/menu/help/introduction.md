# Hub Address について

Hub Address は、職員情報および組織情報をすばやく検索・参照するための Web アプリケーションです。  
CSVファイルからデータを取り込み、必要な情報を即座に見つけ出すことができます。

## 1. アプリの画面構成

Hub Address は、画面左から右に向かって以下の3つのパネルで構成されています。

- **メニュー**
  - アプリ全体の操作（検索キーワードのクリア・データの全件削除・ヘルプ）を行います。
- **職員情報 (Staff)**
  - 職員の氏名、所属、役職、メールアドレスなどの情報を検索・表示します。
- **組織情報 (Division)**
  - 組織の場所、部局・課・係、内線・外線・FAX番号などを検索・表示します。

## 2. 基本的な使い方の流れ

1. 初期状態ではデータが登録されていないため、**職員情報**および**組織情報**の各パネルからCSVファイルをアップロードします。
2. データが取り込まれると、一覧が表示されます。
3. パネル上部の検索フィールドにキーワードを入力して、目的の情報を絞り込みます。
4. 一覧から該当する項目をクリックすると、情報がクリップボードにコピーされ、他方のパネルの検索フィールドに自動的に連携されます（クロス検索）。

## 3. データの管理

Hub Address のデータはブラウザのローカルデータベース（IndexedDB）に保存されます。  
CSVデータを取り込むことで、オフライン環境でも高速に検索を行うことが可能です。  
不要になったデータは、メニューの削除機能から一括で消去できます。
