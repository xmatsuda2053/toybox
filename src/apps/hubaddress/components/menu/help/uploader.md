# データアップロード (UPLOADER) の使い方

Hub Address を初めて利用する際や、データを全件削除した後は、CSVファイルをアップロードしてデータを登録する必要があります。

## 1. アップロードの手順

1. データが未登録の場合、各パネルの中央に点線で囲まれたアップロードエリアが表示されます。
   - 職員情報パネルには「職員情報」のアップロードエリアが表示されます。
   - 組織情報パネルには「組織情報」のアップロードエリアが表示されます。
2. アップロードエリアをクリックして、用意したCSVファイルを選択します。（または、CSVファイルをエリア内に直接ドラッグ＆ドロップします。）
3. ファイルが読み込まれると、自動的にデータが取り込まれ、一覧画面が表示されるようになります。

## 2. CSVファイルのフォーマット

アップロードするCSVファイルは、1行目がヘッダー（キー）として扱われます。

- **職員情報用 CSV** の例
  - `staffId,nameKj,nameKn,div,post,mail1,mail2`
  - 職員番号、漢字氏名、カナ氏名、所属、役職、内部メールアドレス、外部メールアドレス
- **組織情報用 CSV** の例
  - `place,div1,div2,div3,post,other,tel1,tel2,fax,remark`
  - 場所、部局、課、係、役職、その他、内線、外線、FAX、備考

### 2.1 アップロード時の注意点

データを取り込む際、空行は自動的に除外されます。  
アップロードが完了すると、ブラウザのローカルデータベースにデータが保存されるため、次回以降のアクセス時には再アップロードの必要はありません。
