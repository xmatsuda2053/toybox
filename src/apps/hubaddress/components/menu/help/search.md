# 検索機能 (SEARCH) の使い方

各情報パネルの上部には、データを絞り込むための検索フィールドが用意されています。

## 1. 基本的な検索

検索フィールドにキーワードを入力すると、リアルタイムでリストが絞り込まれます。

- **職員情報の検索**
  - 氏名、カナ氏名、所属、役職など、入力されたキーワードに合致する項目が絞り込まれます。
- **組織情報の検索**
  - 場所、部局名、課名、役職、内線番号など、入力されたキーワードに合致する項目が絞り込まれます。

## 2. 検索キーワードのクリア

検索フィールドのキーワードを削除するには、以下の方法があります。

1. **手動で削除:** キーボード操作で直接入力内容を消去します。
2. **メニューの Clear ボタン:** 画面左端のメニューから Clear（消しゴムアイコン）をクリックすると、職員情報と組織情報の両方の検索キーワードが同時にクリアされます。

## 3. クロス検索

一方のリスト内のアイテムをクリックすると、自動的にそのアイテムのテキストが他方の検索フィールドに入力され、関連情報が絞り込まれる機能があります。  
詳しくは「ビューアー (VIEWER) の使い方」を参照してください。
