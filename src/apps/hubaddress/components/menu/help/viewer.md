# ビューアー (VIEWER) の使い方

ビューアーは、アップロードされた職員情報および組織情報を一覧表示する領域です。

## 1. 一覧の見方

リストにはテーブル形式でデータが並んでおり、各列には対応する情報が表示されます。  
データ量が多い場合でもスムーズにスクロールできるよう最適化されています。

- **職員情報リスト**
  - 職員番号、氏名、所属、役職、メールアドレスなどが表示されます。
- **組織情報リスト**
  - 場所、所属（部局/課/係）、内線・外線番号、FAX、備考などが表示されます。

## 2. アイテムのクリックアクション

リスト内に表示されている各項目（セル）をクリックすると、以下の2つのアクションが同時に実行されます。

### 2.1. クリップボードへのコピー

クリックした項目のテキスト内容が自動的にクリップボードにコピーされます。  
メールアドレスや内線番号、氏名などを他のアプリケーション（メールやチャットツールなど）に貼り付ける際に便利です。

### 2.2. クロス検索の実行

クリックした内容が検索対象として有効な場合、自動的にもう一方のパネルの検索キーワードとして入力・追加されます。

- **例:** 職員情報のリストから特定の「所属名」をクリックすると、そのテキストが組織情報の検索フィールドに設定され、該当する組織の情報が瞬時に絞り込まれます。
- **例:** 組織情報のリストから「役職名」をクリックすると、そのテキストが職員情報の検索フィールドに設定され、該当する職員の一覧が表示されます。

この機能を活用することで、職員から所属する部署の詳細を調べたり、部署から所属する職員を探したりする作業がシームレスに行えます。
