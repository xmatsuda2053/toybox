# リスト (LIST) の使い方

LIST は、登録されているタスクを一覧表示するメインセクションです。  
サイドバーで選択したラベルやクイックアクセスの条件に基づいて、表示内容が絞り込まれます。

## 1. 基本的な操作

1. 画面中央にある **「LIST」** セクションにタスクの一覧が表示されます。
2. タスクをクリックすると、そのタスクが「選択中」の状態になり、右側の詳細パネルにタスクの内容が表示されます。
3. 選択中のタスクには **「>」** のアイコンが表示されます。

## 2. タスクのフィルタリング

タスクが多い場合は、セクション上部の検索フィールドにキーワードを入力することで、タスク一覧をリアルタイムに絞り込むことができます。  
入力欄右端の「×」ボタンをクリックすると、フィルタをクリアして全件表示に戻ります。

> **ショートカット:** `Alt + Shift + F` を押すと、ラベルとクイックアクセスの選択をすべて解除し、検索フィールドにフォーカスを当てることができます。

## 3. タスクの並び順と年度グループ

タスクは **年度ごとにグループ化** されて表示されます。  
各年度グループには、そのグループに属するタスクの件数が表示されます。

## 4. タスクアイテムの見方

各タスクアイテムには以下の情報が表示されます。

- **ステータスアイコン**
  - タスクの進行状態を示します。
  - 開始待ち / 対応中 / 完了 の3種類があります。
- **タスク名**
  - タスクの名称です。
- **お気に入りアイコン**
  - クリックするとお気に入り（ブックマーク）の登録・解除を切り替えることができます。
- **期限日**
  - タスクに設定された期限日が表示されます。
  - 期限を過ぎたタスクは赤色でハイライトされます。
  - 期限が3日以内に迫っているタスクは黄色でハイライトされます。
- **ラベル名**
  - タスクが属するラベルの名前が表示されます。
  - ラベル未設定の場合は「未分類」と表示されます。

## 5. タスクの追加

タスクの追加は、LABELS セクションにラベルが1件以上登録されている場合にのみ行えます。

1. LIST セクションのヘッダー右側にある **「＋」ボタン** をクリックします。
   - **ショートカット:** `Alt + Shift + T` を押しても同様にダイアログを開くことができます。
2. 「Add Task」ダイアログが開きます。
3. 以下の項目を入力します。
   - **タスク名** — タスクの名称を入力します。
   - **期限日** — タスクの期限日を選択します。初期値は当日の日付です。
   - **年度** — タスクを分類する年度を選択します。初期値は現在の年度です。
   - **ラベル** — タスクを分類するラベルを選択します。初期値は現在選択中のラベルです。
4. **「追加」** ボタンをクリックすると、タスクが登録されます。
   - 登録後、そのタスクは自動的に選択中の状態になります。
