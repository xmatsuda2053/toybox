# ジャーナル (JOURNAL) の使い方

JOURNAL は、LIST で選択したタスクに紐づく記録を管理するセクションです。  
**Log** と **Note** の2つのタブで構成されています。

## 1. 基本的な操作

1. LIST でタスクをクリックして選択すると、JOURNAL セクションに選択したタスクのログとノートが表示されます。
2. タスクを切り替えると、表示内容は自動的に新しいタスクのものに更新されます。
3. Note タブに内容が入力されている場合、タブ名にマーク（インジケーター）が表示されます。

## 2. Log タブ

Log タブでは、タスクに関する時系列の作業記録（ログ）を管理します。  
各ログは Markdown 形式で記述できます。

### 2.1. ログの追加

1. Log タブ下部にある **「＋（Add）」ボタン** をクリックします。
   - **ショートカット:** `Alt + Shift + L` を押しても新規ログを追加できます。
2. ログ一覧の末尾に新しいログエントリが追加され、自動的に最下部にスクロールします。
3. 追加されたエディタに内容を入力します。入力内容は自動的に保存されます。

### 2.2. ログの見方

各ログアイテムには以下の情報が表示されます。

- **作成日時**
  - ログが作成された日時（`yyyy-MM-dd HH:mm` 形式）が表示されます。
- **相対時刻**
  - 作成日時から現在までの経過時間が表示されます（例：「3時間前」）。
- **Markdown エディタ**
  - ログの内容を Markdown 形式で編集できます。入力内容は自動的に保存されます。

### 2.3. ログのフィルタリング

ログが多い場合は、Log タブ上部の検索フィールドにキーワードを入力することで、一覧をリアルタイムに絞り込むことができます。  
入力欄右端の「×」ボタンをクリックすると、フィルタをクリアして全件表示に戻ります。

### 2.4. ログのスクロール

Log タブ下部のボタンでログ一覧をスクロールできます。

- **「↑（Scroll Up）」ボタン** — 一覧の先頭（最古のログ）までスクロールします。
- **「↓（Scroll Down）」ボタン** — 一覧の末尾（最新のログ）までスクロールします。

### 2.5. ログの削除

1. 削除したいログの右上にある **メニューアイコン（三本線）** をクリックします。
2. ドロップダウンメニューから **「削除」** を選択します。
3. 「Confirm Delete」ダイアログが開きます。
4. **「削除」** ボタンをクリックすると、ログが完全に削除されます。操作を取り消す場合は **「キャンセル」** をクリックします。

> **注意:**  
> ログの削除は取り消せません。

## 3. Note タブ

Note タブでは、タスクに関する自由なメモ（ノート）を1件管理します。  
ログと異なり、追加・削除はなく、常に1つのエディタが表示されます。

1. Note タブをクリックして切り替えます。
2. エディタに内容を Markdown 形式で自由に入力します。
3. 入力内容は自動的に保存されます。
4. ノートに内容が入力されている場合、Log タブから見てもタブ名にマークが表示されます。
