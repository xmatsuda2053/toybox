# ラベル (LABELS) の使い方

LABELS は、タスクをカテゴリーごとに分類・整理するためのナビゲーションセクションです。  
ラベルを選択すると、そのラベルが設定されているアイテムのみがメイン画面に表示されます。

## 1. 基本的な操作

1. 画面のサイドバー（ナビゲーション領域）にある **「LABELS」** というセクションに移動します。
2. 表示されたラベル名をクリックすると、その状態が「選択中」となり、メイン画面のリストがそのラベルで絞り込まれます。
3. もう一度クリックすると「選択解除」となり、絞り込みが解除されます。

## 2. ラベルのフィルタリング

ラベルが多い場合は、セクション上部の検索フィールドにキーワードを入力することで、ラベル一覧をリアルタイムに絞り込むことができます。  
入力欄右端の「×」ボタンをクリックすると、フィルタをクリアして全件表示に戻ります。

## 3. ラベルの追加

1. LABELS セクションのヘッダー右側にある **「＋」ボタン** をクリックします。
2. 「Label Editor」ダイアログが開きます。
3. ラベル名を入力します。
4. **「保存」** ボタンをクリックすると、ラベルが登録されます。
   - 保存後、そのラベルは自動的に選択中の状態になります。

## 4. ラベルの編集

1. 編集したいラベルにカーソルを合わせ、右端に表示される **メニューアイコン（三本線）** をクリックします。
2. ドロップダウンメニューから **「プロパティ」** を選択します。
3. 「Label Editor」ダイアログが開き、現在のラベル名が入力済みの状態で表示されます。
4. ラベル名を編集し、**「保存」** ボタンをクリックすると変更が反映されます。

## 5. ラベルの削除

1. 削除したいラベルにカーソルを合わせ、右端に表示される **メニューアイコン（三本線）** をクリックします。
2. ドロップダウンメニューから **「削除」** を選択します。
3. 「Confirm Delete」ダイアログが開きます。
4. **「削除」** ボタンをクリックすると、ラベルが完全に削除されます。操作を取り消す場合は **「キャンセル」** をクリックします。

> **注意:**
> ラベルの削除は取り消せません。
> また、そのラベルが設定されていたアイテムは「未分類」として扱われるようになります。
