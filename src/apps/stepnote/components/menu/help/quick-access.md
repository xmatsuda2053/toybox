# クイックアクセス (QUICK ACCESS) の使い方

クイックアクセスは、登録されているタスクやログ及びノートの中から、特定の条件に一致するものをすばやく絞り込んで表示するためのナビゲーションメニューです。

## 1. 基本的な操作

1. 画面のサイドバー（ナビゲーション領域）にある **「QUICK ACCESS」** というセクションに移動します。
2. 「QUICK ACCESS」の右側にある矢印アイコン（折りたたみボタン）をクリックすると、メニューの開閉（展開/折りたたみ）を切り替えることができます。
3. 表示された各項目の名称をクリックすると、その状態が「選択中」となり、メイン画面のリストがその条件で絞り込まれます。
4. もう一度クリックすると「選択解除」となり、絞り込みが解除されます。

## 2. メニュー項目の一覧と機能

クイックアクセスから選択できる絞り込み条件は以下の通りです。  
複数の項目を組み合わせた絞り込みが可能な場合があります。

### 2.1. マーク・分類

- **お気に入り**
  - ブックマーク（お気に入り）に登録したアイテムのみを表示します。
  - よく確認する重要な情報をすぐに見つけたい場合に便利です。
- **未分類**
  - 特定のラベルが設定されていない、分類待ちのアイテムを表示します。

### 2.2. 期限・スケジュール

- **期限切れ** （赤色のアイコンが目印です）
  - 設定された期限を過ぎてしまったアイテムを表示します。
  - 優先的に対応が必要なタスクの確認に使用します。
- **期限間近** （黄色のアイコンが目印です）
  - 設定された期限が迫っているアイテムを表示します。
  - スケジュール管理や直近のタスク確認に使用します。

### 2.3. 進捗・ステータス

- **完了**
  - すでに対応が完了した（ステータスが「完了」の）アイテムを表示します。
- **対応中**
  - 現在作業を進めている（ステータスが「対応中」の）アイテムを表示します。
- **開始待ち**
  - まだ作業に取り掛かっていない（ステータスが「開始待ち」の）アイテムを表示します。
