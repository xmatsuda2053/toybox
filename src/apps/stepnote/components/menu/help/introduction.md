# Step Note について

Step Note は、タスクの進捗と記録を一元管理するための Web アプリケーションです。  
タスクをラベルで分類し、各タスクに対してログ（作業記録）とノート（メモ）を蓄積していくことができます。

## 1. アプリの画面構成

Step Note は、画面左から右に向かって以下の5つのパネルで構成されています。

- **メニュー**
  - アプリ全体の操作（表示切替・データのインポート／エクスポート）を行います。
- **ナビゲーション（QUICK ACCESS / LABELS）**
  - 表示するタスクをすばやく絞り込むための条件を選択します。
- **リスト（LIST）**
  - 絞り込み条件に合致するタスクを一覧表示します。
- **タスク（TASK）**
  - 選択したタスクの詳細情報を表示・編集します。
- **ジャーナル（JOURNAL）**
  - 選択したタスクに紐づくログとノートを管理します。

## 2. 基本的な使い方の流れ

1. **LABELS** でラベルを作成し、タスクのカテゴリーを定義します。
2. **LIST** でタスクを追加します。タスクにはラベル・期限日・年度を設定します。
3. **LIST** でタスクをクリックすると、**TASK** と **JOURNAL** にその内容が表示されます。
4. **TASK** でタスク名・期限日・関係者・説明を編集します。
5. **JOURNAL** の Log タブに作業記録を追記します。必要に応じて Note タブにメモを残します。
6. 作業が進んだら **TASK** の **「Mark as」** ボタンからステータスを更新します。

## 3. ショートカットキー一覧

| ショートカット    | 機能名             | 説明                                                            |
| :---------------- | :----------------- | :-------------------------------------------------------------- |
| `Alt + Shift + E` | ナビゲーション開閉 | ナビゲーションパネルと LIST パネルの表示／非表示を切り替えます  |
| `Alt + Shift + T` | タスク新規追加     | タスク追加ダイアログを開きます                                  |
| `Alt + Shift + L` | ログ新規追加       | 選択中のタスクに新しいログを追加します                          |
| `Alt + Shift + F` | 全検索モード切替   | 全ての絞り込みを解除し、LIST の検索フィールドにフォーカスします |

## 4. データの管理

Step Note のデータはブラウザのローカルデータベース（IndexedDB）に保存されます。  
データのバックアップや別環境への移行には、メニューのインポート／エクスポート機能を使用してください。

- **エクスポート** — 現在のデータをファイルに書き出します。
- **インポート** — エクスポートしたファイルからデータを復元します。

> **注意:**  
> インポートを実行すると、現在のデータは上書きされます。事前にエクスポートでバックアップを取得することを推奨します。
