# メニュー (MENU) の使い方

メニューは、画面の左端に配置されているアイコン群です。  
アプリ全体の表示切り替えやデータの管理を行うための機能を提供します。

## 1. メニュー項目の説明

上から順に以下のボタンが配置されています。

### 1.1. Explore（表示切り替え）

一番上のファイルアイコンのボタンです。  
クリックすると、画面左側のナビゲーション（QUICK ACCESS / LABELS）とリスト（LIST）の表示／非表示を切り替えることができます。  
タスクの詳細やジャーナルの編集に集中したい場合（Thin Mode）に便利です。

> **ショートカット:** `Alt + Shift + E` を押しても同様に表示を切り替えることができます。

### 1.2. Import（データのインポート）

上から2番目のアップロードアイコンのボタンです。  
エクスポートしたバックアップデータ（ファイル）を読み込み、アプリに復元します。  
クリックするとファイル選択ダイアログが開くので、インポートしたいファイルを選択してください。  
処理が完了すると「Complete」ダイアログが表示されます。

> **注意:**  
> インポートを実行すると、現在のデータは上書きされる場合があります。  
> インポートする前に、念のため現在のデータをエクスポートしてバックアップしておくことを推奨します。

### 1.3. Export（データのエクスポート）

上から3番目のダウンロードアイコンのボタンです。  
現在アプリに登録されている全てのデータ（タスク、ログ、ノート、ラベル等）をファイルとして書き出し、ダウンロードします。  
定期的なバックアップや、別の端末へのデータ移行に使用します。

なお、下記の時間帯に自動エクスポートを実行します。  
ブラウザを起動していない場合など想定し、1日に複数回実行します。

| 時間帯 | 説明                                       |
| :----- | :----------------------------------------- |
| 08:30  | 業務開始時点のバックアップです。           |
| 12:00  | 午前中の作業分のバックアップです。         |
| 17:15  | 当日の作業分を含むバックアップです。       |
| 20:00  | 残業した場合を想定してのバックアップです。 |

### 1.4. Help（ヘルプ）

一番下の「？」アイコンのボタンです。  
クリックすると、このヘルプ画面（マニュアル）が表示されます。  
各機能の詳しい使い方やショートカットキーの一覧を確認できます。
