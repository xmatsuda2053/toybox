# タスク (TASK) の使い方

TASK は、LIST で選択したタスクの詳細を表示・編集するセクションです。  
**Summary** と **Property** の2つのタブで構成されています。

## 1. 基本的な操作

1. LIST でタスクをクリックして選択すると、TASK セクションに選択したタスクの内容が表示されます。
2. 各入力欄を編集すると、内容は自動的に保存されます。
3. ヘッダー右側の **「Mark as」ボタン** をクリックすると、タスクのステータスを変更できます。

## 2. ステータスの変更

**「Mark as」** ドロップダウンから変更先のステータスを選択します。  
選択できるステータスは以下の3種類です。

- **開始待ち**
  - まだ作業を開始していない状態です。
- **対応中**
  - 現在作業を進めている状態です。
- **完了**
  - 作業が完了した状態です。

> ステータスを変更すると、変更内容がログに自動記録されます。

## 3. Summary タブ

Summary タブでは、タスクの主要な情報を編集します。

- **タスク名**
  - タスクの名称を入力します。
  - タスク名の左側にはステータスアイコンが表示されます。
  - **「Copy ID」ボタン** をクリックすると、Markdown リンク形式のタスク ID とタスク名をクリップボードにコピーできます。
- **期限日**
  - タスクの期限日を日付ピッカーで設定します。
- **関係者**
  - タスクに関連する担当者の情報を入力します。
  - **所属** — 担当者の組織名・部署名を入力します。
  - **氏名** — 担当者の氏名を入力します。
  - **電話番号** — 担当者の電話番号を入力します。
- **タスク説明**
  - タスクの詳細説明を Markdown 形式で記述できます。

## 4. Property タブ

Property タブでは、タスクの分類情報や管理用メタデータを編集します。

- **ID**
  - タスクに自動付与された一意の識別番号です。変更はできません。
- **作成日**
  - タスクが作成された日時です。変更はできません。
- **更新日**
  - タスクが最後に更新された日時です。変更はできません。
- **年度**
  - タスクが属する年度を選択します。変更するとリスト上の年度グループも更新されます。
- **ラベル**
  - タスクが属するラベルを選択します。「未分類」を選択するとラベルなしの状態になります。

## 5. タスクの削除

タスクの削除は Property タブから行います。誤操作防止のため、2段階の操作が必要です。

1. Property タブを開きます。
2. **「このタスクを削除可能とする」** スイッチをオンにします。
3. **「Delete」** ボタンが有効になるのでクリックします。
4. 「Confirm Delete」ダイアログが開きます。
5. **「削除」** ボタンをクリックすると、タスクが完全に削除されます。操作を取り消す場合は **「キャンセル」** をクリックします。

> **注意:**  
> タスクの削除は取り消せません。  
> タスクに紐づくログやノートもあわせて削除されます。
