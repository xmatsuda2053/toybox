description: ステージングされた変更のみから日本語のコミットメッセージを生成してコミットします。
---

# Steps

1. **Get Staged Diff Only**
   - 以下のコマンドを実行し、ステージングされている変更**のみ**を取得してください。未ステージングのファイルや作業スペースの状態は完全に無視してください。
   - `git diff --cached`

2. **Generate Message in Japanese**
   - 取得した `git diff --cached` の内容だけを元に、コミットメッセージを生成してください。
   - **厳格なルール**:
     - プレフィックス（`feat:`, `fix:` 等）やファイル名、パス、変数名などの英単語は**そのまま英語で維持**すること。
     - それ以外の説明文・要約は、**必ずすべて日本語**で記述すること。
     - フォーマット: `<type>: <日本語の要約>`
   - **使用タイプ**: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`

3. **Commit**
   - 生成したメッセージで `git commit -m "<generated_message>"` を実行するか、実行前にユーザーに確認を求めてください。
